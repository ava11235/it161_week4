Notes

Forms are used whenever information is collected from website visitors. Information from a form is sent in name/value pairs.

Each form control is given a name, and the text the user types in or the values of the options they select are sent to the server.

A user fills in the form, and once they press "Submit" button, the form info is sent to the server.

The server creates a new page to send back to the browser based on info received.

There are various form controls (widgets), used for adding text, passwords, making choices on the form, etc.
